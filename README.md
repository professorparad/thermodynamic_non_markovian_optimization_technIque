# Hybrid Optimization Framework (v3.9) - Phase 1

## Structure
- `src/`: Core logic including the Hybrid Optimizer.
- `scripts/`: Benchmarking and utility scripts.
- `test/`: Unit and integration tests.
- `plots/` & `outputs/`: Artifact storage.

## Usage
Run the main entry point:
```bash
python phase1/main.py
```

Run benchmarks:
```bash
python phase1/scripts/run_benchmarks.py
```
